Github绑定域名试错

上次使用GitHub Page是几年前了，当时还不能直接添加CNAME和Enforce HTTPS。

来自网上的教程提示，绑定域名的时候，要在域名DNS页面添加4个GITHUB的IP，并写上自己的域名。

like this：

A example.com 185.199.108.153
A example.com 185.199.109.153
A example.com 185.199.110.153
A example.com 185.199.111.153

但是实际上只要先ping一下你的github page

ping example.github.io

得到一个ip地址，并把它添加到 A 里面就行了，而且名称写@就好，不需要写具体域名

like this:

A @ 你艾特出来的域名

TTL可以设置的短一点儿（比如：600s），然后等一会儿就OK啦
